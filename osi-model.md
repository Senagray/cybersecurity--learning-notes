# OSI Model

## 1. Physical Layer
Responsible for physical data transmission.

## 2. Data Link Layer
Handles MAC addresses and frame transmission.

## 3. Network Layer
Responsible for IP addressing and routing.

## 4. Transport Layer
Provides TCP and UDP communication.

## 5. Session Layer
Manages communication sessions.

## 6. Presentation Layer
Formats and encrypts data.

## 7. Application Layer
Provides services for applications.
